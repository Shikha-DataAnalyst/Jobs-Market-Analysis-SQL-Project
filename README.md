# Job-Market-Analysis-Project-SQL

## Project Overview:
This SQL project analyzes job market data to uncover hiring trends, salary patterns, remote work opportunities, and in-demand technical skills across different job roles and companies.

The project uses structured SQL queries to transform raw job posting data into actionable business insights that support workforce and market analysis.

## Business Problem:
With the rapid growth of technology-driven hiring, companies and professionals need better visibility into:
- Salary trends
- Remote work opportunities
- Skill demand
- Hiring patterns across locations
- Employment types and compensation structures

**This project was built to analyze job posting data and identify meaningful trends using SQL-based analysis.**

## Dataset Information:
The dataset contains job-related information such as:
- Job title
- Company name
- Location
- Salary details
- Employment type
- Remote work availability
- Search terms
- Job descriptions
- Salary ranges and standardized salary metrics

## Database Schema Highlights
The `jobs_data` table includes key fields such as:
- `title`
- `company_name`
- `location`
- `schedule_type`
- `work_from_home`
- `salary_avg`
- `salary_min`
- `salary_max`
- `salary_standardized`
- `search_term`

These fields were used to perform salary, hiring, and remote work analysis.

## Business Questions Solved:
- Which job roles offer the highest average salaries?
- Which companies are hiring most frequently?
- What percentage of jobs offer remote work?
- Which locations provide higher salary opportunities?
- Which job types dominate the market?
- What are the most in-demand technical skills?
- How do hourly and yearly salaries compare across roles?

## Key Insights:
- Remote job opportunities were available across multiple technical roles.
- Data Analyst, Software Engineer, and Machine Learning roles showed strong market demand.
- Salary ranges varied significantly based on job type and employment structure.
- Technical skills such as SQL, Tableau, Excel, and AI-related technologies appeared frequently in job descriptions.
- Full-time positions represented a major share of job postings.
